# Bandishare Blog

Bandishare Blog là trang phụ dạng tĩnh được tạo bằng `index.html`, dùng để chia sẻ tin tức game, cập nhật phiên bản, mẹo chơi và các bài viết ngắn xoay quanh game Android.

Trang này không xây dựng theo kiểu website tải APK như site chính, mà tập trung vào nội dung blog/tin tức để hỗ trợ nhận diện thương hiệu và điều hướng người dùng về Bandishare khi cần xem thêm bài chi tiết.

## Nội dung chính

- Tin mới về game Android
- Cập nhật Minecraft, Roblox và các game phổ biến
- Mẹo chơi, hướng dẫn nhanh
- Gợi ý game mô phỏng, chiến thuật, casual
- Bài viết chia sẻ xu hướng và trải nghiệm game

## Cấu trúc file

## Cách chạy trên GitHub Pages

1. Vào **Settings** của repository.
2. Chọn **Pages**.
3. Ở mục **Build and deployment**, chọn:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Bấm **Save**.
5. Chờ GitHub tạo link Pages.

## Mục đích sử dụng

Repository này dùng để tạo một trang phụ nhẹ, dễ index, dễ chia sẻ và phù hợp làm blog vệ tinh nội dung cho hệ thống Bandishare.
